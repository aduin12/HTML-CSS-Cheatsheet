HTML Element Cheatsheet

Some notes before reviewing the table. Remember that all elements below require a closing tag e.g <h1>This is a header</h1> unless otherwise specified.

Element Name HTML Tag Element Description
Headings <h1>, <h2>, <h3>, <h4>, <h5>, <h6> These tags define HTML headings. <h1> is used for main headings, followed by <h2> headings, then the less important <h3>, and so on.
Paragraph Element <p> This element represents a paragraph. Paragraphs are usually represented as blocks of text.
Ordered and Unordered Lists <ol> or <ul> These list elements are used to designate the beginning of a list. Using the <ol> tag will create a numbered list. Using the <ul> tag will create a list with bullet points.
List Element <li> The list item element can be used to create list items inside ordered lists <ol> and unordered lists <ul>.
Link Element <link> The link tag defines the relationship between the current document and an external resource. The link element is an empty element, it contains attributes only and a closing tag is not needed. The link tag is most often used to link to external style sheets. Here is an example of a link tag in html: <link rel="stylesheet" href="styles.css">.
Video Element <video> The video element embeds a media player for video playback. Here is an example of the video element being used in html: <video src ="test-video.mp4" controls> Video not supported </video>. The src attribute will contain the URL to the video. Adding the controls attribute will display video controls in the media player. The content inside the opening and closing tag is shown as a fallback in browsers that don't support the element.
Image Element <img> The image tag is used to embed an image in an HTML page. Images are not techincally inserted into a web page; images are linked to web pages. The image tag creates a holding space for the reference image. The image tag has two required attributes: src and alt. src specifies the path to the image and alt specifies an alternate text for the image, if the image for some reason cannot be displayed.
Emphasis Element <em> The emphasis element emphasizes text. Browsers will usually italicize the emphasized text by default.
Strong Element <strong> The strong element highlights important, serious, or urgent text and browsers will normally render this highlighted text in bold by default.
Div Element <div> The div element is used as a container that divides an HTML document into section and is short for "division". Div elements can contain flow content such as headings, paragraphs, links, images, etc.
Closing tag </__> An HTML closing tag is used to denote the end of an HTML element. Most HTML elements need a closing tag.
Body Element <body> The body element represents the content of the HTML document. Content inside body tags are rendered on the web browsers. There can only be one body element in a document.
Span Element <span> The span element is an inline container for text and can be used to group text for styling purposes. However as span is a generic container to separate pieces of text from a larger body of text, its use should be avoided if a more semantic element is available.
Head Element <head> Contains the metadata/information for the document. Usually where the CSS stylesheet is linked and the title of the website is listed.
Title Element <title> Defines a title for the document
The a Tag <a> The <a> tag defines a hyperlink, which is used to link from one page to another. The most important attribute of the <a> element is the href attribute, which indicates the link's destination. By default, links will appear in all browsers as underlined and blue when unvisited, underlined and purple as visited, and underlined and red when active. Here is an example of the <a> tag in HTML: <a href="https://www.coolmathgames.com">Visit the best website on the internet!</a>
